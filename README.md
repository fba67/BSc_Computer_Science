## Undergraduate Projects — University of Tehran

This repository contains selected projects from my undergraduate studies, covering areas such as compiler design, systems programming, web development, software engineering, and machine learning.

While these projects were developed earlier in my academic career, they demonstrate fundamental skills in algorithms, system design, and software development that form the basis of my current work in machine learning.

## Project Categories

- **Compiler** – Parsing and lexical analysis
- **Debugger** – Low-level debugging in assembly
- **Data Structures & Algorithms** – Graph algorithms, scheduling, and problem solving
- **GUI Applications** – Interactive puzzle games
- **Database Applications** – Desktop application with SQL Server backend

**Note:** These projects are being uploaded retrospectively, so commit timestamps may not reflect their original development dates.
