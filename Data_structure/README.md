# Data Structures Projects

This folder contains several undergraduate projects focused on classical data structures and algorithms.

## Included Projects

- APSP – All-Pairs Shortest Path implementation (Dijkstra-based)
- K_Formula – Graph-to-string encoding and decoding using a custom star-based representation
- mastermind – Implementation of the Mastermind game
- PERT – Project scheduling using PERT algorithm

These projects demonstrate core algorithmic thinking, graph algorithms, and problem-solving techniques. They are originally implemented in C# as part of undergraduate coursework.
