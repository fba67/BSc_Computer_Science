# K-Formula (Graph Encoding and Reconstruction)

## Description
Implementation of a custom "K-Formula" representation for directed graphs, including conversion from an adjacency matrix to a string-based encoding and reconstruction of the graph from that encoding.

## Concepts
- Graph representation
- Adjacency matrices
- String-based encoding of graph structure
- Recursive parsing and reconstruction algorithms

## Notes
Includes both encoding (graph → K-Formula) and decoding (K-Formula → graph) functionality.
