# Minesweeper (GUI Game Implementation)

## Description
Implementation of the classic Minesweeper game with a graphical user interface, where the player reveals cells while avoiding hidden bombs.

## Concepts
- GUI development (Windows Forms)
- Event-driven programming
- Grid-based game design
- Flood-fill algorithm using a stack
- State management and user interaction

## Notes
Includes dynamic grid generation, random bomb placement, and recursive expansion of empty cells based on neighboring values.
