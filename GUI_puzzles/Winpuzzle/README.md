# WinPuzzle (Sliding Puzzle Game)

## Description
Implementation of a sliding puzzle game with a graphical user interface, where the user rearranges tiles to reach a target configuration.

## Concepts
- GUI development (Windows Forms)
- Event-driven programming
- Grid-based state management
- Game logic and user interaction

## Notes
Includes dynamic creation and handling of button-based tiles and user-triggered actions within a desktop application.
