# GUI Puzzles

## Description
This folder contains a collection of interactive puzzle games implemented with graphical user interfaces, focusing on user interaction and event-driven programming.

## Included Projects
- WinPuzzle – Sliding puzzle game implemented using Windows Forms
- Minesweeper – Classic minesweeper-style grid-based puzzle game

## Concepts
- GUI development (Windows Forms)
- Event-driven programming
- Grid-based game design
- State management and user interaction

## Notes
These projects demonstrate implementation of interactive applications with graphical interfaces, complementing algorithmic and systems-oriented work in other parts of the repository.
